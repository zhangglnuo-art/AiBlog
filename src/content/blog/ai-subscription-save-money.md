ChatGPT $20、Claude $20、Midjourney $10、Cursor $20……一个个看着不贵，加起来一个月轻松上百刀。这篇讲讲怎么把 AI 订阅的钱花在刀刃上，以及那些**容易踩的坑**。

## 一、先砍掉「其实没在用」的

最大的浪费不是买贵了，而是**买了不用**。动手前先做一件事：

- 打开每个订阅，回想**过去 30 天你真正打开过几次**
- 一个月用不到 5 次的，直接停掉——需要时再订回来，损失很小

**先用免费版跑满一个月再决定要不要付费**，比一上来就订更划算。功能差异可参考[GPT-5、Claude、Gemini 怎么选](/blog/gpt-vs-claude-vs-gemini/)。

## 二、年付 vs 月付：能省 15%～20%

多数 AI 服务的年付比月付便宜 **15%～20%**（相当于白送两个月）。

**但年付有前提**：
- ✅ 你已经连续用了 2～3 个月，确定是刚需
- ❌ 刚订一周就冲年付——万一不合适，退款通常很麻烦

**建议**：新工具先月付试水，稳定成为主力后再转年付。

## 三、错峰订阅：按项目订，用完就退

很多人不需要「常年持有」所有工具。更聪明的用法是**按需订、按月轮换**：

- 这个月做视频 → 订 Runway，做完退掉
- 下个月做设计 → 订 Midjourney

AI 订阅几乎都是**按月自动续费、可随时取消**，取消后通常还能用到当期结束。用这种方式，一年能省下不少。

## 四、轻度使用？API 可能更便宜

如果你一个月只用几十次，**按量付费的 API 往往比 $20 包月更划算**——用多少付多少。反过来，天天重度使用的人，包月才是省钱的那个。

想走 API 路线，见[OpenAI / Claude API 充值攻略](/blog/openai-claude-api-topup-guide/)（记得设消费上限，避免超支）。

## 五、关于「合租 / 共享账号」——请谨慎

网上到处是「AI 会员合租」，价格确实诱人，但风险要说清楚：

- **多数平台的服务条款禁止账号共享**，被判定共享可能**封号**，钱和账号一起没
- **隐私风险**：共享账号意味着别人**可能看到你的对话历史**，工作资料、私人内容都在里面
- **不稳定**：车主随时可能改密码、跑路，出问题**没有退款渠道**
- **风控牵连**：同一账号多地登录容易触发风控

**建议**：真要省钱，优先用前面几招（砍冗余、年付、错峰、API）。用工作资料或重要内容的场景，**别用合租账号**。

## 六、退订避坑

这是最容易「白花钱」的环节：

- **取消 ≠ 立刻失效**：取消后一般能用到当期结束，别急着重复取消
- **一定要在续费日前取消**：很多人是「扣款后才想起来」
- **确认取消成功**：看到「订阅将于 X 月 X 日结束」的提示才算数，截图留证
- **删 App ≠ 退订**：卸载应用不会停止扣款
- **注意试用转正**：免费试用到期会自动转付费，试用期就该设好提醒

## 七、用虚拟卡管理多个订阅

订阅一多，最怕的是「**忘了哪张卡在扣什么钱**」。用[虚拟信用卡](https://rdvcc.com)管理有几个实际好处：

- **一服务一卡**：哪个订阅在扣钱一目了然，要停就停那张
- **额度可控**：卡里只放够用的余额，避免忘记取消被连扣半年
- **不绑定个人主卡**：不用把常用银行卡暴露给多个海外平台
- **解决付款被拒**：国内双币卡常被海外 AI 平台拒付，虚拟卡本身也是这个问题的解法

## 小结

省钱的顺序其实很清晰：**先砍掉不用的 → 主力工具转年付 → 非常用的按需轮换 → 轻度使用改用 API**。合租看着便宜，但封号和隐私风险不值得。再配上虚拟卡把订阅管起来，每月账单就不会再失控了。
