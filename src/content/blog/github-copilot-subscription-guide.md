GitHub Copilot 是微软和 GitHub 官方出品的 **AI 编程助手**，也是这个品类的开山鼻祖——「AI 帮你写代码」这件事就是它带火的。它以插件形式装进你现有的编辑器（VS Code、JetBrains、Neovim 都支持），补全、对话、改代码一步到位。这篇讲清它的免费与付费区别、国内怎么订阅付款，以及和 Cursor 怎么选。

## Copilot 能做什么

- **代码补全**：写一半自动续上，注释写清楚需求它能生成整个函数
- **Copilot Chat**：在编辑器里直接问「这段代码什么意思」「帮我修这个报错」
- **Agent 模式**：给它一个任务，它能跨文件改代码、跑命令、自己验证
- **代码评审**：PR 里让 Copilot 先过一遍，抓低级错误

它最大的优势是**不用换编辑器**——装个插件，现有工作流原地升级。

## 免费版和付费版的区别

Copilot 有免费档，轻度用户可以先白嫖：

| | Free | Pro（$10/月） | Pro+（$39/月） |
| --- | --- | --- | --- |
| 代码补全 | 每月有限次数 | 无限 | 无限 |
| Chat 对话 | 每月有限次数 | 充足额度 | 更高额度 |
| 高级模型 | 受限 | 可选主流旗舰模型 | 更多前沿模型 |
| Agent 任务 | 受限 | 支持 | 额度更高 |

轻度尝鲜用 Free 就够；**每天写代码的人上 Pro，$10/月是主流 AI 编程工具里最便宜的一档**。Pro+ 适合重度依赖 Agent 和前沿模型的用户。各档额度调整频繁，**以官网定价页为准**。

## 学生和开源维护者：免费

这是很多人不知道的：**通过 GitHub Education 认证的学生和教师，以及活跃的开源项目维护者，可以免费用 Copilot Pro**。在校生用教育邮箱去 GitHub Education 申请认证即可，几天内通过——有学生身份的先走这条路，一分钱不用花。

## 国内怎么订阅（付款是唯一门槛）

Copilot 的注册和安装都没有门槛：有 GitHub 账号，在 **Settings → Copilot** 里开通即可。卡就卡在付款——

和其他海外 AI 服务一样，Copilot 走的是海外支付通道，**国内双币卡经常被拒**，报错类似 `Your card was declined`。原因是卡的发行地区和账单信息被风控判定为高风险，反复试卡还可能连累账号。

更稳的做法是用一张地区信息可控的[虚拟信用卡](https://rdvcc.com)，以 [RDVCC](https://rdvcc.com) 为例：

1. 注册并用 USDT 充值，几分钟开出一张 Visa/Mastercard 虚拟卡
2. 在 GitHub **Settings → Billing** 里添加这张卡，账单地址按开卡地区信息填
3. 回到 **Settings → Copilot** 选择 Pro 订阅，扣款成功立即生效

同一张卡还能顺手覆盖 ChatGPT、Claude、Cursor 等其他订阅，不用每个平台单独折腾。各支付方式的完整对比见[海外 AI 订阅支付方式全对比](/blog/ai-subscription-payment-methods/)。

## Copilot 还是 Cursor？

这是被问得最多的问题，简单说：

- **Copilot**：插件形态，不换编辑器、上手零成本，$10/月，适合想在现有 IDE 里加个强力助手的人
- **Cursor**：独立编辑器，代码库理解和多文件改动更强，$20/月，适合愿意迁移工作流换更强能力的人

预算有限先试 Copilot（还有免费档），追求上限选 Cursor——详细对比见[AI 编程工具横评](/blog/ai-coding-tools-comparison/)和 [Cursor 订阅指南](/blog/cursor-subscription-guide/)。

## 常见问题

**Q：免费额度用完了会怎样？**
补全和对话会停到下个月恢复。真到了每月都用光的程度，说明你已经是 Pro 的目标用户了。

**Q：公司电脑能用吗？**
注意合规：不少公司对 AI 工具处理内部代码有规定，企业场景应该用公司统一采购的 Business/Enterprise 版，别用个人账号处理公司代码。隐私相关的考虑见[AI 隐私保护指南](/blog/ai-privacy-guide/)。

**Q：订阅后所有编辑器都能用吗？**
可以。订阅绑定 GitHub 账号，VS Code、JetBrains、Neovim 等装上插件登录同一账号即可。

## 小结

Copilot 是 AI 编程工具里**门槛最低的一档**：不换编辑器、有免费档、付费也只要 $10/月，学生还能白嫖。把支付工具提前备好，整个开通流程十分钟搞定——想系统了解还有哪些值得订的工具，见[2026 年海外 AI 工具盘点](/blog/best-overseas-ai-tools-2026/)。
